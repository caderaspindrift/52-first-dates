# Colophon {#anchor-colophon}

The primary text font is 
[Bellefair](https://fonts.google.com/specimen/Bellefair)
<br/>
by [Nick Shinn and Liron Lavi Turkenich](https://github.com/shinntype/bellefair).

The header font is 
[Cherry Swash](https://fonts.google.com/specimen/Cherry+Swash)
by Nataliya Kasatkina.

The card font is
[Noto Sans](https://fonts.google.com/specimen/Noto+Sans)
by [Google](https://github.com/googlefonts/noto-fonts).

The emoji font is
[Noto Emoji Color](https://github.com/googlefonts/noto-fonts)
by [Google](https://github.com/googlefonts/noto-emoji/).

Additional icons are in
[Noto Emoji](https://github.com/googlefonts/noto-fonts)
by [Google](https://github.com/googlefonts/noto-emoji/).

The dyslexic font is
[Open Dyslexic](https://opendyslexic.org/)
by [Abbie Gonzalez](https://github.com/antijingoist/opendyslexic).

The low vision font is
[APHont](https://www.fontspace.com/aphont-font-f4926)
by [American Printing House](https://www.aph.org/).

[vim](https://www.vim.org/),
[GNU make](https://www.gnu.org/software/make/manual/make.html),
[Pandoc](https://pandoc.org/), 
[Lua](https://www.lua.org/),
and
[Prince](https://www.princexml.com/)
<br/>
were used in the creation of this manuscript.
